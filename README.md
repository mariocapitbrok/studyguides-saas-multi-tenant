# SaaS + Multi-tenant

This repository contains the StudyGuides content and configuration for `SaaS + Multi-tenant`.

## Files

- `docs/` topic documentation
- `mkdocs.yml` site configuration
- `Makefile` local preview and build commands
- `topic.yaml` machine-readable topic metadata

## Local Development

1. `make pull`
2. `make serve`
3. Open `http://127.0.0.1:8000`

## Notes

This guide focuses on tenant isolation, SaaS platform design, operations, and delivery patterns.
